Project Overview
Flawlessly Blended is a professional makeup artistry business founded in 2024 by Masego Dooka, a student from TUT. What started as a small scale beauty service has grown into a recognized brand known for high quality glam and modern styling more than anything its dedication. The mission is to provide clients with professional, comfortable and creative beauty experiences for weddings, graduations and matric dances, making people feel seen and the highlight of special occasions. This project involves creating a professional 5 page digital portfolio to showcase services and streamline the booking process via a WhatsApp integrated system, this will make things better and easier for both the user and the owner when it comes to bookings .

Technical Stack & Functionality

- HTML5: Used for semantic structure (Headers, Main, Section, and Footer tags)
-	CSS3: Implemented for the "Glam" aesthetic, using Flexbox and Grid for a mobile responsive layout
-	JavaScript (ES6): Developed for form validation and the WhatsApp API integration, allowing users to send booking data directly from the browser to the artist's phone
-	Version Control: Managed via GitHub to track development milestones and code integrity

Website Goals and Objectives
-	Establish Professionalism: Transition from social media booking to a dedicated brand website
-	Services: Provide transparent pricing for Natural, Soft and Full Glam packages
-	Booking Experience: Implement a functional booking form that sends details directly to the artist
-	Build Trust: Display a high resolution gallery of past client work
-	Mobile Optimization: Ensure the site is fully responsive for clients booking on the go

 Sitemap & Navigation
-	index.html: Landing page contains the Hero section and "About Us" narrative
-	gallery.html: A responsive image grid featuring past makeup artistry
-	pricing.html: Transparent services pricinf (Natural, Soft and Full Glam)
-	booking.html: A dynamic form requiring user input (Name, Date, Service)
-	contact.html: Direct links to Instagram and WhatsApp contact points
